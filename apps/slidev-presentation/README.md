# Slidev Presentation

OOAD AI 專案的簡報套件，使用 Slidev 製作。

## 線上簡報

🚀 **Live Demo**: [https://ayugioh2003.github.io/ooad-ai/](https://ayugioh2003.github.io/ooad-ai/)

此簡報透過 GitHub Actions 自動部署到 GitHub Pages。

> 📝 更新時間：2025-06-27

## 開發

```bash
# 在根目錄執行
pnpm dev:slidev

# 或者在此目錄執行
pnpm dev
```

## 建置

```bash
# 在根目錄執行
pnpm build:slidev

# 或者在此目錄執行
pnpm build
```

## 匯出

```bash
# 在根目錄執行
pnpm export:slidev

# 或者在此目錄執行
pnpm export
```
